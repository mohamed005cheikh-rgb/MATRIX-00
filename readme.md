# HSBTAK - ISI Special Edition

منصة تحويل وحساب أنظمة العد مصممة بأسلوب سينمائي حديث لطلاب المعهد العالي للمعلوماتية (ISI).

## المميزات الجديدة:
- **خلفية سينمائية:** استخدام فيديو `promo.mp4` كخلفية تفاعلية.
- **واجهة زجاجية:** تصميم (Glassmorphism) مريح للعين.
- **أيقونات متحركة:** تحسين الوضوح والحركة عند التفاعل.
- **ثنائي اللغة:** دعم كامل للعربية والفرنسية بنقرة واحدة.
- **تواصل مباشر:** ربط حصري برقم الواتساب.

## طريقة التشغيل:
1. ضع `index.html`, `style.css`, `script.js` في مجلد واحد.
2. تأكد من وجود فيديو باسم `promo.mp4` في نفس المجلد.
3. افتح `index.html` في المتصفح.
